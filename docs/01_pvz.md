# Задача 1. Пункты выдачи Ozon

**Цель:** из xlsx с текстовыми адресами получить чистый датасет всех ПВЗ Красноярска с координатами.

## Вход

- `data/raw/active_points_Other.xlsx` (36 958 строк по всей РФ)

## Выход

- **Файл:** `data/processed/pvz_krasnoyarsk.csv`
- **Колонки:**
  - `id` — уникальный идентификатор точки
  - `address_raw` — исходный адрес из xlsx
  - `address_clean` — нормализованный адрес
  - `latitude` — широта (WGS84)
  - `longitude` — долгота (WGS84)
  - `working_hours` — часы работы
  - `geocoding_quality` — качество геокодации (точный адрес / улица / город)
- **Ожидаемый объём:** ~420 строк
- Отдельный лог-файл или колонка про проблемные случаи (не удалось геокодировать, нестандартные адреса)

## Этапы работы

### 1. Фильтрация
Прочитать файл (шапка на строке 7), оставить только `Регион = Красноярский край` и `Город = Красноярск`. Проверить, что получается ~420 строк.

### 2. Чистка адресов
Адреса грязные: `Россия Красноярский Красноярск Карла Маркса 139` → `Красноярск, ул. Карла Маркса, 139`.

Что нужно сделать:
- убрать дубли города (`Красноярский Красноярск` → `Красноярск`)
- привести сокращения к единому виду (`ул.`, `пр-т`, `пер.`)
- убрать лишние пробелы (двойные → одинарные)
- проставить запятые между компонентами адреса

### 3. Геокодация
Превратить текстовые адреса в координаты. Два варианта:

**Nominatim (OSM)**
- бесплатный
- лимит 1 запрос/сек
- по русским адресам может путаться
- на 420 адресов ~10 минут

**Яндекс.Геокодер**
- точнее для России
- бесплатно до 1000 запросов/день
- нужен API-ключ (регистрация в Яндекс.Облаке)

### 4. Валидация
Проверить, что все координаты попали в границы города. Те, что за пределами — пересмотреть вручную или пометить как «не удалось».

## Инструменты

- `pandas` — работа с таблицами
- `requests` — запросы к API геокодеров
- `geopy` — обёртка над Nominatim: [документация geopy](https://geopy.readthedocs.io/)
- [Яндекс.Геокодер API](https://yandex.ru/dev/geocoder/doc/ru/)
- `folium` — быстрая проверка (нанести точки на карту и посмотреть глазами)

## Подводные камни

- **Грязные адреса.** Некоторые такие, что ни один геокодер не справится — будьте готовы 5–10 штук обработать вручную.
- **Политика Nominatim.** Есть обязательный User-Agent, иначе банят — [политика использования](https://operations.osmfoundation.org/policies/nominatim/).
- **Сохраняйте промежуточные результаты.** Не геокодировать все 420 адресов в цикле по одному — если упадёт, не начинать с нуля. Лучше батчами, с сохранением в файл после каждого батча.

## Критерий готовности

Итоговый CSV загружается в Folium, все ~420 точек видны на карте города, ни одна не улетела в Красноярский край или за рубеж.
